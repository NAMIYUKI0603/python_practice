# テキストマイニングWebアプリ

ブラウザ上で動作する軽量なテキストマイニング用アプリケーションです。
PDFファイルやテキストファイルを読み込み、出現頻度の高い単語（名詞）を抽出して、棒グラフやワードクラウドで直感的に可視化します。

## 主な機能
- **ファイル解析**: PDFおよびテキスト（UTF-8推奨）の自動読み込み
- **形態素解析**: Janomeを用いた名詞の抽出、不要な単語（ストップワード）や1文字単語の除外
- **可視化ツール**: 
  - 単語の出現頻度上位20語の棒グラフ化およびデータテーブル表示
  - ワードクラウドの生成（解像度選択対応：標準/高画質/超高画質）
- **マスク機能**: 白黒画像をアップロードすることで、指定したシルエットの形にワードクラウドを生成

## ローカル環境での実行方法

事前にPython（3.8以上推奨）をインストールしておいてください。

1. このリポジトリをダウンロード（または `git clone`）します。
2. ターミナル（コマンドプロンプトやPowerShell等）でフォルダを開きます。
3. 必要なライブラリをインストールします。
   ```bash
   python -m pip install -r requirements.txt
   ```
4. Streamlitアプリを起動します。
   ```bash
   python -m streamlit run app.py
   ```
   
> **Note**: アプリを終了する場合は、ターミナルで `Ctrl + C` を押してください。

## 日本語フォントについての注意点
Linuxベースの環境（Streamlit Community Cloudを含む）やMac等で実行した際、日本語フォントがインストールされていないとワードクラウド・matplotlibのグラフが文字化け（豆腐化）する可能性があります。
その場合は、アプリと同じフォルダ内に日本語のTrueTypeフォント（例: `font.ttf` コピー）を配置するか、実行環境に日本語フォントパッケージをインストールしてください。Windows環境では標準フォントを自動で読み込みます。
