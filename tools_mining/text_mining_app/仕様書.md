# テキストマイニングWebアプリ 仕様書

## 1. アプリケーションの概要
本アプリケーションは、PDFやテキスト（TXT）ファイルから文章を抽出・解析し、どのような単語が頻繁に使われているかを直感的に可視化する「テキストマイニングツール」です。
特別な知識がなくても、ブラウザ上の簡単な操作で文章の特徴を掴むことができます。

## 2. 動作環境・前提条件
* **提供形式**: Webブラウザ上で動作するローカルアプリケーション（現在はお客様のパソコン内でのみ動作）
* **対応ブラウザ**: Google Chrome, Microsoft Edge, Safariなどのモダンブラウザ
* **対応ファイル形式**:
  * `.txt`（一般的なテキストファイル / 文字コード: UTF-8推奨）
  * `.pdf`（文書ファイル / 画像化されたPDFではなく、テキストデータが含まれているもの）

## 3. 主な機能と画面構成

画面は大きく「左側の設定パネル」と「右側の結果表示パネル」に分かれています。

### 3.1 左サイドバー（操作・設定パネル）
* **ファイルアップロード**:
  * 分析したいPDFまたはTXTファイルをドラッグ＆ドロップ、またはクリックして選択します。
* **除外する単語（ストップワード）の設定**:
  * 「する」「いる」といった、文章の特徴を掴む上で不要な一般的な単語をあらかじめ除外リストとして指定できます。
  * デフォルトでいくつか設定されていますが、自由にカンマ区切り（例: `私,今日,非常` ）で追加・編集が可能です。
* **分析実行ボタン**:
  * 設定完了後、このボタンを押すことで文章の読み込みと解析処理が開始されます。

### 3.2 右メインエリア（結果表示パネル）
分析実行後、以下の順番で結果が表示されます。

1. **テキストのプレビュー**: アップロードされたファイルから抽出した文章の先頭500文字を確認できます。
2. **出現頻度上位20語 (棒グラフ)**: 文章中で最も多く使われている単語（名詞）トップ20を、左側の棒グラフで視覚的に表示します。
3. **出現頻度上位20語 (データテーブル)**: 棒グラフと同じ結果を、具体的な「単語」と「出現回数」の表形式で正確に確認できます。
4. **ワードクラウド**: 文章に含まれる単語を、出現頻度が高いほど文字を大きく、低いほど文字を小さくして一枚の画像として表現します（最大100単語）。

## 4. 処理の裏側（テキストマイニングの仕組み）
本アプリでは、以下の手順で文章を解析しています。

1. **テキスト抽出**: PDFやTXTファイルから文字データだけを取り出します。
2. **不要文字の削除（クレンジング）**: URL、カッコなどの記号、数字、改行などを自動的に取り除き、きれいな文章にします。
3. **形態素解析（単語の切り出し）**: 文章を「単語」ごとに分解し、その中から意味を持つ「名詞」だけを抽出します。
4. **フィルタリング**: 「1文字だけの単語（例: あ, い, う）」や、左サイドバーで設定した「除外する単語」を取り除きます。
5. **集計・可視化**: 残った単語の数を数え、グラフやワードクラウドを作成します。

## 5. 使用上の注意点と制限事項（重要）

初心者の方向けの重要な注意事項です。ご利用前に必ずご確認ください。

### 5.1 解析できるファイルについて制限
* **画像化されたPDFは解析不可**: スキャナーで紙を読み込んだだけの「文字をなぞって選択できないPDF（画像PDF）」からは、テキストを抽出できないためエラーになります。
* **ファイルサイズと文字数の上限**:
  * 推奨ファイルサイズは **5MB以下** です。
  * 数百ページに及ぶ巨大なPDFや、数百万文字のテキストを入力すると、パソコンのメモリ（処理能力）が不足し、アプリが強制終了（フリーズ）するリスクがあります。
  * 目安として、まずは「数十ページ程度のPDF」や「数万文字程度のテキスト」からお試しください。

### 5.2 解析結果（マイニング結果）の特性・限界
* **完璧な単語分割はできません**: 日本語の自動解析ツール（形態素解析エンジン）を使用しているため、特殊な専門用語、造語、人名などが誤った位置で区切られることがあります。（例: 「テキストマイニング」が「テキスト」と「マイニング」に分かれる等）。
* **「名詞」のみを対象としています**: 動詞（走る、食べる）や形容詞（美しい、大きい）は、今回は特徴語から除外される仕様となっています。
* **意味合い（ポジティブ・ネガティブ）は判定しません**: 単に「どの単語がたくさん出たか（出現頻度）」だけをカウントするため、その単語が良い意味で使われたか、悪い意味で使われたかの感情分析（センチメント分析）は行いません。

### 5.3 セキュリティ・情報の取り扱い
* 現在のバージョンは「お客様のパソコン内」でのみ処理が完結するため、アップロードしたテキストやPDFが**外部のインターネット（クラウド等）に送信されることはありません**。機密情報を含む文書でも安全に解析できます。（※将来的にWeb上に一般公開した場合はこの限りではありません）。
* アプリを終了すると、アップロードしたファイルや解析結果のグラフは**すべて消去（初期化）**されます。結果を残したい場合は、グラフ画像を右クリックして保存するか、スクリーンショットを撮ってください。

## 6. トラブルシューティング（よくあるエラーと対処法）

* **Q: 「テキストを抽出できませんでした」と表示される**
  * A: 選択したPDFが画像PDFである可能性が高いです。Word等から直接PDF保存された別のファイルでお試しください。
* **Q: 「有効な単語が抽出されませんでした」と表示される**
  * A: 文章が極端に短い場合や、記号・数字ばかりのデータ、あるいは全ての単語が「除外する単語」に指定されている場合に発生します。
* **Q: グラフの日本語が「□□□（豆腐）」のように文字化けする**
  * A: 使用しているパソコンのOS（Windows以外のMac等）で、指定した日本語フォントが見つからない場合に起こります。現在の仕様はWindows環境に最適化されています。
